Eternity Commands

Commands can be invoked by prefixing with the global prefix . or one of custom prefixes set by the server owner (check out prefix commands).

You can always list all commands by typing .help in chat, or if you want to see it's usage and description - type .help (command here) eg. .help welcomer

If you want to ask us something - Join Eternity's server

Basic commands

Command Aliases Description Usage
userinfo (none) Displays information about user. .userinfo [user]
ping (none) Checks bots latency. .ping
help (none) Displays command listing or usage, description and aliases. .help
invite (none) Shows Enterniy's invite url. .invite
serverinfo (none) Shows server's information. .serverinfo
